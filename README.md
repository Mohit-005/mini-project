# Ubuntu Portfolio

An Ubuntu-themed portfolio website built with Next.js.

## Features

- Ubuntu desktop interface
- Interactive applications
- Responsive design

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js
- TailwindCSS
- React
